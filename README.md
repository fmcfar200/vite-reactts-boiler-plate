Boilerplate Vite, React, TS, Redux (React-Router-DOM & React-Bootstrap) application

## Requirements
 - Vite requires Node.js version >=12.0.0.
 - NPM

## Setup
  1. Pull down the repo
  2. run `npm install`
  3. run `npm run vite`
